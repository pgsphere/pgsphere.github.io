Source code repository
PgSphere source code hosting in done on Github, the central repository being https://github.com/postgrespro/pgsphere . The easiest way to start development on pgSphere is forking the repository on Github or cloning it into a local directory tree. Also, consider subscribing to the mailing list to get in touch with other developers and with users.
